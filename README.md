# corona-flask
A simple web app for showing the total cases of corona all around the world, cities

# Installation:
- git clone https://github.com/krishpranav/corona-flask
- cd corona-flask
- sudo chmod +x *
- pip3 install venv
- python3 -m venv env
- source env/bin/activate
- python3 app.py

- then head to http://0.0.0.0:5000/
